# Student Management System in C

## Description
A simple menu-driven Student Management System built using C. It allows users to add, display, and search student records.

## Features
- Add student details (roll number, name, marks)
- Display all students
- Search student by roll number

## Technologies Used
- C Programming
- Structures
- Functions

## How to Run
1. Compile the program:
   gcc main.c -o student
2. Run the program:
   ./student
